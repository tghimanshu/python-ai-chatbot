# AI ChatBot

A python based AI Chatbot

## Description

A python based Chatbot which provides uses AI to respond to chats

## Getting Started

### Dependencies

- python 3.9+
- flask

### Installing

- Create a virtual Env

  ```
  python -m venv venv
  ```

- initialize the env

  ```
  venv\Sscripts\activate
  ```

- install the libraries

  ```
  pip install -r requirements.txt
  ```
